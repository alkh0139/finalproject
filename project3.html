Blog Platform

Developed a blog platform using Django and integrated user authentication. The platform offers user registration, author profiles, article creation and editing, and commenting functionality.

Technologies Used: Django, Python, HTML, CSS, SQLite.

Achievements: Built an easy-to-use interface that allows authors to create and manage content seamlessly.

Challenges Faced: The primary challenge was implementing rich text editing capabilities for article creation. Ensuring proper data security for user accounts was also a critical aspect of the project.
