E-Commerce Website

Built a fully functional e-commerce website using React and Node.js. The website offers user registration and authentication, a comprehensive product catalog, efficient shopping cart functionality, and secure payment processing.

Technologies Used: React, Node.js, Express, MongoDB, Stripe for payments.

Challenges Faced: One of the key challenges was implementing real-time inventory tracking to ensure accurate product availability. Additionally, optimizing performance for a large number of products was crucial to provide a smooth user experience.
